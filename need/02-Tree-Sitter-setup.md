# Tree-Sitter 配置困难

- **来源**: Reddit r/emacs，用户 u/bong (2026)
- **链接**: https://www.reddit.com/r/emacs/comments/1jdr8j6/treesitter_usage_or_alternatives/

## 需求描述

用户从 Vim 转到 Emacs，发现 Tree-Sitter 配置过于复杂。官方文档说要重新编译 Emacs 开启 Tree-Sitter 支持，用户觉得为了一个以前是包的功能而重新编译太麻烦。

## 关键痛点

- 配置流程不清晰，文档分散
- 需要分别安装 grammars，过程繁琐
- built-in treesit 和第三方 elisp-tree-sitter 容易混淆
- 用户更希望开箱即用
